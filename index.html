Abelian Sandpile Visualization

The Abelian Sandpile Model is a cellular automaton that demonstrates self-organized criticality. Each cell contains a number of "grains" (0-3). When a cell reaches 4 or more grains, it "topples", distributing grains to its neighbors, creating beautiful fractal patterns.
